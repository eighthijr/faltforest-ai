# Landing Page Structure (AIDA + Proof)

Target: penjual non-teknis.
Bahasa: Indonesia sederhana.

## Section structure + copy

1. **Hero (Attention + Interest)**
   - "Buat halaman jualan yang bikin orang langsung ingin beli"
   - CTA utama: **Mulai Gratis Sekarang**
2. **Problem (Interest)**
   - "Sudah capek promosi, tapi orang cuma lihat lalu pergi."
3. **Solution (Desire)**
   - "Kami bantu kamu bikin halaman jualan yang jelas, menarik, dan bikin calon pembeli lebih percaya."
4. **Demo (Desire)**
   - 3 langkah singkat: isi data produk -> pilih gaya -> langsung siap dipakai.
5. **Benefits (Desire)**
   - Fokus pada manfaat bisnis: lebih cepat jualan, lebih dipercaya, pesan lebih jelas.
6. **Proof (Proof)**
   - Angka hasil + testimoni singkat pengguna.
7. **CTA (Action)**
   - "Buat Halaman Jualan Saya Sekarang"
8. **Footer**
   - Link penting dan kalimat penutup.

## Usage

```tsx
import { LandingPage } from './components/landing';
```
